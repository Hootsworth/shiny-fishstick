From DOM Chaos to Semantic Stability

XHR Interception & API Upgrades

Monitors network boundaries during crawling and maps browser clicks to direct, high-speed API requests.

Heuristic DOM Scoring

Ranks CSS target selectors (`data-testid` > `id` > `name` > `class`) to survive interface updates and element changes.

State Machine Discovery

Tracks sequential page dependencies to translate browser actions into a verified Finite State Machine.

Auth & Storage Serialization

Automatically detects authorization forms and snapshots cookies, localStorage, and sessionStorage snapshots.

Token Propagation

Fuzzy-correlates intercepted bearer headers with storage resources, injecting auth into direct REST routines automatically.

Auto-Generated MCP Server

Auto-compiles a Model Context Protocol server exposing targets directly as tools to LLMs (Cursor/Claude Code).

    Install via pip

    Import and Run
    
                                
    Initialize the browser session and call compiled site actions:
    
                                
    from shiny_fishstick_sdk import ShinyFishstickSiteSDK

# Initialize SDK client
sdk = ShinyFishstickSiteSDK("http://localhost:8001")
sdk.start(headless=True)

try:
    sdk.login("admin@example.com", "password123")
    sdk.search_products("shoes")
    sdk.add_to_cart("product-42")
    sdk.checkout()
finally:
    sdk.close()

    2
    
                            
    
                                
    Integrate in TypeScript
    
                                
    Initialize the browser page and call direct API upgrades:
    
                                
    import { ShinyFishstickSiteSDK } from 'shiny-fishstick-sdk';

async function run() {
    const sdk = new ShinyFishstickSiteSDK();
    await sdk.start(true); // headless = true
    
    try {
        await sdk.login('admin@example.com', 'password123');
        await sdk.search_products('shoes');
        await sdk.add_to_cart('product-42', 1);
        await sdk.checkout();
    } finally {
        await sdk.close();
    }
}
run();
